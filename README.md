# SatCore 🛰️

> Mission control dashboard for satellite health monitoring, telemetry analysis, and pass scheduling powered by real orbital data.

## Overview

SatCore is an open-source satellite mission control and telemetry analysis platform built with Python and Streamlit. Designed to simulate real ground segment operations, it provides tools for monitoring spacecraft health, analyzing telemetry data, and scheduling ground station passes.

## Planned Features

- **Satellite Health Dashboard** — real-time subsystem status monitoring
- **Telemetry Analyzer** — anomaly detection and performance trend analysis
- **Pass Scheduler** — ground station pass prediction using real TLE data
- **Orbit Visualizer** — 3D orbital track visualization

## Tech Stack

- [Python 3.10+](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
